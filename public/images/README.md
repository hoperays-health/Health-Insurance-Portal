# Images Folder

## Add Your Beach Background Image

Place your beach family image here as:
- **hero-beach.jpg**

This image is used as the background for:
- Authentication pages (login, register, etc.)
- Onboarding flow pages

### Recommended Image Specs
- Format: JPG or PNG
- Dimensions: At least 1920x1080px
- Orientation: Landscape
- File size: Optimized for web (< 500KB recommended)

### Don't have an image?
You can use any beach/ocean image or family photo. The design will still work without it - it will just show a solid color background.

### Alternative
If you don't have a suitable image, you can:
1. Download a free stock photo from [Unsplash](https://unsplash.com/s/photos/beach)
2. Use a solid color by removing the `beach-background` class from the layouts
